üìñ About the Project

The UAP/UFO/NHI Disclosure Database is a comprehensive, community-driven resource documenting unidentified aerial phenomena, non-human intelligence encounters, government disclosures, and humanity's attempts to communicate with potential extraterrestrial intelligence.

This project began as a personal chronology and has evolved into an interactive database featuring over 1,000 documented events spanning from ancient times to the present day. The database includes government hearings, FOIA releases, military encounters, whistleblower testimony, scientific studies, and historical sightings.

üîß Technical Details

This project is built with modern web technologies and designed to be fast, accessible, and work offline as a Progressive Web App (PWA).

Technology Stack

  • Frontend: Vanilla JavaScript, HTML5, CSS3
  • Visualization: D3.js for timeline, Leaflet.js for maps
  • Data Format: Markdown (chronology.md)
  • Hosting: GitHub Pages
  • Internationalization: Custom i18n system supporting 15+ languages
  • PWA: Service worker for offline functionality

Features

  • Interactive D3.js timeline with zoom and pan
  • Leaflet.js world map with clustered markers
  • Detailed chronology list view with markdown rendering
  • Advanced filtering by time period, category, and geography
  • Multilingual support (15+ languages including binary and Morse code)
  • Progressive Web App with offline support
  • Responsive design for mobile and desktop
  • Dark theme optimized for readability
